Second-Hand Car Price Prediction System

This project uses Linear Regression and Gradient Boosting algorithms to predict the price of second-hand cars. The system takes into account various features like car make, year, mileage, engine specifications, and other factors, providing an accurate price estimate. Developed with Flask for the web framework, the project integrates a database for storing car details, and allows users to upload car images along with the car's features. The system also includes a recommendation feature to suggest similar cars based on user preferences.
